# ARPENT — Démo courtier immobilier · NEXTIWEB Studio

Site vitrine de démonstration pour un **courtier immobilier résidentiel** (Montréal & Rive-Sud), réalisé par **NEXTIWEB Studio**.

> ⚠️ **Contenu 100 % fictif** (nom de marque, coordonnées, avis, chiffres). Page en `noindex` — démonstration uniquement.

## Aperçu

- **Direction artistique :** « console / tableau de bord » hi-tech 2026.
- **Palette :** kaki militaire · blanc cassé · jaune pastel.
- **Typographie :** Space Grotesk · Inter · Space Mono.
- **Pièce maîtresse :** un sélecteur de **personas** interactif (premier achat, vente, achat-vente simultané, investisseur/plex, nouvel arrivant) qui révèle les douleurs spécifiques de chaque profil et la réponse du courtier.
- **Bilingue** FR / EN avec sélecteur de langue.
- Maison en plan d'architecte « À vendre » dans le hero, conformité **Loi 25**, accessibilité (skip-link, `prefers-reduced-motion`), responsive.

## Structure

```
index.html        ← version française
en/index.html     ← version anglaise
```

Chaque page est **autonome** : tout le CSS et le JavaScript sont intégrés, aucune dépendance externe (hors Google Fonts). Ouvrez simplement `index.html` dans un navigateur.
